        Config preset (add to config.yaml, then restart ECH):
      

      
# Asterisk PBX — real hardware via AMI
- type: asterisk
  name: pbx
  ami_host: 127.0.0.1      # Asterisk host (localhost if on same machine)
  ami_port: 5038
  ami_username: ech
  ami_secret: changeme
  local_extension: "101"   # Your ATA/phone extension
  page_target: "SIP/101"   # Extensions to page (& separated for multiple)
  context: from-internal
  caller_id: "ECH <100>"
  log_inbound: true
  log_outbound: true
  # Screen phone (add later):
  # screen_extension: "102"
  # screen_push_url: "http://192.168.1.50/CGI/Execute"

# --- OR for testing without hardware ---
- type: mock_asterisk
  name: pbx-mock
  interval_sec: 60
  local_extension: "101"

      

        Asterisk AMI requires manager.conf with ECH user having system,call,log,originate permissions.
        See Asterisk AMI docs.
